Diferença entre innerText, innerHTML e TextContent:

O que é o innerText?

O innerText foca no conteúdo de texto renderizado. Por exemplo, quando você utiliza o innerText para ler o conteúdo de um elemento, ela retorna o texto como aparece na tela. Ignorando tags HTML e também não incluindo texto oculto por estilos CSS.

O que é o innerHTML?

O innerHTML lê tanto a marcação HTML quanto o conteúdo de texto do elemento. Isso significa que, ao usá-la para definir o conteúdo de elementos, você pode incluir tags HTML e o navegador as renderizará corretamente.

O que é o TextContent?

O TextContent também ignora todas as tags HTML e retorna apenas o texto. Mas diferente do innerText, ele retorna o texto estando renderizado na tela ou não.
